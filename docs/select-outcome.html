4.3 Select Outcome

There are two types of outcomes:

  • Summary measures: When selected, the result will be a single measure for the entire dataset. For instance when Summary Measure - Life Expectancy (Summary Measure) is selected the result is a single life expectancy for the given for the population. (ref:Summarycalcstrat) When stratifications are selected, the summary measure will be given for each strata. For instance when Summary Measure - Life Expectancy and Stratification - Sex are selected, then two life expectancy measures will be given one for males and one for females.
  • By row measures: When selected, the result will be the measurement for each individual (e.g.Â row) in the dataset.

More detials of how the calculations are performed can be found in Key Concepts (Chapter 4).

Click on box beside the calculation name under Select initial calculations. A single outcome or multiple outcomes may be selected.
